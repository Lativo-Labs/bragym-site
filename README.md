# Gym Landing Page

A single-page website built for a friend's gym ( which I go to ) as a learning project.

## What I used
- Antigravity (Claude / Gemini)
- Google Stitch + MCP
- HTML / CSS
- Render for deployment

## Why I built it
First hands-on project using AI-assisted development tools. 
Goal was to get familiar with the workflow from idea to deployed page.
